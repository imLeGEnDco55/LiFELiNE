## 2024-05-18 - Auth Strong Password Requirement\n**Vulnerability:** Weak password requirement on signup (min length 6, no complexity requirement) potentially allowing brute force and weak account security.\n**Learning:** The validation logic for password was too permissive during new account creation, and failed to check against standard password strength constraints.\n**Prevention:** Centralized password validation rules using `isStrongPassword` explicitly applied only to the registration flow to ensure new accounts have robust passwords while supporting existing users on login.
